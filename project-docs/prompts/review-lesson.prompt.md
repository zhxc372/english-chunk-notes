# Prompt — 审核 Lesson JSON

你是一名严格的英语学习内容审核员。请审核下面的 lesson JSON。

## 审核目标

判断它是否适合中文母语者用于：

1. 写作表达
2. 主题词块积累
3. 听力识别
4. 闪卡复习
5. 技术英语 / 考试英语迁移

## 审核维度

请逐项检查：

1. 文章是否自然？
2. 文章是否太空泛？
3. 词块是否值得学习？
4. 是否混入太简单的普通词？
5. 是否有太长或太生僻的词块？
6. 中文含义是否准确？
7. 搭配是否自然？
8. 例句是否可以朗读？
9. technical_sentence 是否准确？
10. exam_sentence 是否生硬？
11. retelling_prompt 是否能引导输出？
12. exam_tags 是否合理？
