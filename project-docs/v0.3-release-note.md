# v0.3 Release Note

## 日期
2026-05-26

## 核心变化

### 架构重构
- 从「主题词块站」升级为「文章驱动词块站」
- 新增 Category / Article / Chunk / ArticleChunkOccurrence 数据模型
- Article 与 Chunk 分离，M:N 关系

### 新功能
- **划词查询**：选中文字即可查词，3级Provider（当前文章→全局词块→外部词典）
- **Web Speech 朗读**：浏览器原生TTS，支持语速/音调/声音设置
- **全局词块库**：独立于文章，支持变形形式和近义表达
- **首页仪表盘**：学习方法论 + 最新文章 + 分类入口
- **分类页**：按栏目浏览文章

### 移除/废弃
- ❌ 不再依赖 MP3 文件（保留旧文件但新功能不使用）
- ❌ 不使用 Edge TTS
- ❌ 不在 Web Worker 中调用 TTS

### 数据迁移
- 3个旧版lesson已迁移为v0.3格式（education-learning, software-engineering, environment-climate）
- 迁移脚本：`scripts/migrate-lessons-to-v03.py`

### 兼容性
- ✅ 旧版主题页面完好
- ✅ 旧版收藏数据自动兼容（v0.2 → v0.3 localStorage迁移）
- ✅ 闪卡和导出功能保留

## 文件统计
- 新增 20+ 文件
- 新增组件：ArticlePage, ChunkLibrary, ChunkDetail, SelectionLookup, TtsButton, TtsSettingsPanel, HomeDashboard, CategoryPage
- 新增工具：browserTts.ts, lightLemma.ts, normalizeText.ts
- 新增composable：useArticles, useChunks, useArticleChunks, useCategories, userState
