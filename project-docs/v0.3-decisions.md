# v0.3 产品决策

## 定位

v0.3 是文章驱动的英语词块学习站。

参考形态是 Engoo Daily News，但增加全局词块库层。

## 本版本要做

1. 建立 Category / Article / Chunk / ArticleChunkOccurrence 数据模型
2. 页面从 Theme/Lesson 改成 Category/Article 思维
3. Chunk 成为全局知识库，不被文章私有化
4. 文章中词块通过 ArticleChunkOccurrence 指向 Chunk
5. 词块变形使用 surfaceForms 与 occurrence.surface 处理
6. TTS 只使用 Web Speech API
7. 划词查询使用 Provider 架构
8. 搜索覆盖文章、词块、例句、中文解释、分类、标签和考试标签
9. 首页改成「方法论 + 学习记录 + 最新文章 + 分类入口」
10. 避免首页加载全部内容

## 本版本不做

1. 不做 MP3 / MP3 下载 / GitHub Actions 生成 MP3
2. 不做 Edge TTS / Web Worker 调 Edge TTS
3. 不做完整 NLP 词形还原
4. 不自动合并语义相近词块
5. 不强依赖外部 API / 不把 API key 放前端
6. 不重写整个项目

## 性能边界

- 首页不加载所有文章正文和完整词块数据
- 分类页只加载文章列表（标题+摘要）
- 文章页按需加载关联词块
- 搜索使用预构建索引

## 数据原则

- Article 保存语境
- Chunk 保存知识
- ArticleChunkOccurrence 保存出现位置和原文变形
- SearchIndex 保存查询入口
- UserProgress 保存学习状态

## 变形原则

- canonical 是标准形
- surface 是原文形式
- surfaceForms 是已知变形
- aliases 是人工确认近义表达
- relatedChunkIds 是相关表达，不自动合并
