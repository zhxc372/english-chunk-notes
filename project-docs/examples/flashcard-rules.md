# Flashcard Rules

每个收藏词块自动生成 4 种闪卡。

## 1. 英译中

正面：

```text
human review
```

背面：

```text
人工审查；人类审查

AI-generated code should still go through human review.
```

## 2. 中译英

正面：

```text
人工审查；人类审查
```

背面：

```text
human review

require human review
go through human review
human review process
```

## 3. 句子填空

正面：

```text

```

背面：

```text
human review
```
