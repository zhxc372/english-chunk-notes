# PRD — English Chunk Notes

## 1. 产品定位

English Chunk Notes 是一个 **考试英语 + 技术英语双轨主题词块文档站**。

它不是英语学习 SaaS，也不是词典，而是一个围绕主题文章、听力音频、词块收藏和闪卡复习建立的个人学习系统。

## 2. 用户问题

目标用户是中文母语者，典型问题：

1. 能看懂英文，但阅读时烦躁，缺少稳定语块。
2. 会中文思考，但英文写作变成直译。
3. 背单词容易迷糊，不能转成主动表达。
4. 听力遇到不熟悉表达时只能听到单个词。
5. 既要备考，也要学习技术英语 / 专业英语。

## 3. 产品目标

第一版目标：

- 用主题文章提供语境。
- 用 edge-tts 批量生成听力音频。
- 从文章中抽取主题词块。
- 为词块生成含义、搭配、例句和复述任务。
- 用户可以收藏词块。
- 收藏内容可以生成闪卡。
- 支持考试标签和技术英语标签。

## 4. 非目标

第一版不做：

- 登录
- 数据库
- 云同步
- 在线 AI 生成
- 自动作文评分
- 口语评分
- 完整考试题库
- 社区和商业化

## 5. 核心用户路径

```text
打开网站
→ 选择主题
→ 阅读主题文章
→ 播放文章音频
→ 查看核心词块
→ 收藏重要词块
→ 进入闪卡页复习
→ 做中译英 / 听写 / 复述
```

## 6. 页面设计

### 6.1 首页

展示：

- 今日推荐主题
- 主题入口
- 技术英语入口
- 考试标签入口
- 我的收藏
- 闪卡训练

### 6.2 主题页

每个主题页包含：

- 主题说明
- 主题文章
- 文章音频
- 核心词块
- 可套作文段落
- 复述问题
- 相关考试标签

### 6.3 词块卡片

每个词块展示：

- 英文词块
- 中文含义
- 常见搭配
- 英文例句
- 中文翻译
- 音频播放
- 收藏按钮
- 标签

### 6.4 收藏页

展示用户收藏的：

- 主题
- 词块
- 句子
- 段落

第一版只要求收藏词块。

### 6.5 闪卡页

根据收藏词块自动生成：

- 英译中
- 中译英
- 句子填空
- 音频听写

## 7. 第一批内容

第一版先做 3 个主题：

1. AI Coding Workflow
2. Education & Learning
3. Work & Career

每个主题：

- 1 篇 150-250 词文章
- 20 个词块
- 20 条词块例句
- 20 条音频
- 5 个复述任务
- 1 个作文段落模板

## 8. 验收标准

第一版必须满足：

- 能展示至少 3 个主题。
- 每个主题能展示文章、音频和词块。
- 用户能收藏词块。
- 刷新后收藏不丢失。
- 收藏词块能生成闪卡。
- 音频文件存在时能播放。
- 没有登录、数据库、后端和在线 AI 依赖。
