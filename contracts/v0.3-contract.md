# v0.3 Contract

## 项目目标

将 english-chunk-notes 升级为文章驱动的英语词块学习站。

## 不可违反约束

1. 不做 MP3。
2. 不做 MP3 下载。
3. 不做 Edge TTS。
4. 不做 Web Worker 调 Edge TTS。
5. 不把词块私有化到文章里。
6. 不把变形形式创建为新 chunk。
7. 不自动合并近义词块。
8. 不在首页加载所有文章正文和完整词块数据。
9. 不把外部 API key 放在前端。
10. 不破坏 v0.2 已有收藏、闪卡、导出能力。

## 必须实现

1. Web Speech 在线朗读。
2. Category / Article / Chunk / ArticleChunkOccurrence。
3. 词块 canonical / surfaceForms / aliases。
4. Article M:N Chunk。
5. 划词查询 Provider 架构。
6. 文章与词块双索引搜索。
7. 首页方法论 + 学习记录 + 最新文章 + 分类入口。
8. 移动端基础可用。

## 数据原则

```text
Article 保存语境；
Chunk 保存知识；
ArticleChunkOccurrence 保存出现位置和原文变形；
SearchIndex 保存查询入口；
UserProgress 保存学习状态。
```

## 变形原则

```text
canonical 是标准形；
surface 是原文形式；
surfaceForms 是已知变形；
aliases 是人工确认近义表达；
relatedChunkIds 是相关表达，不自动合并。
```
